# 📝 Survey Project

사용자가 설문지를 작성할 수 있고 간단한 결과를 볼 수 있는 웹서비스.
Typescript + SpringBoot 기반으로 제작하였고 JWT 인증을 사용하였다.

---

## 🛠 기술 스택
**Frontend**
- React, TypeScript
- Styled-components / CSS Modules

**Backend**
- Java 17, Spring Boot
- Spring Security, JWT
- JPA, MySQL, Redis
